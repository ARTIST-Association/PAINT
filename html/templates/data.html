Database Overview

Calibration, deflectometry, kinematic, weather

Data Repository

Our web page offers an HTML-browsable version of the data from the concentrating solar power plant in Jülich, Germany, with the global power plant id (GPPD) WRI1030197. A programmatic API is provided in our repository as a PyTorch dataset and STAC client.

Data Types

Calibration
RGB photos of the focused sunlight spots of heliostats on dedicated calibration receivers.
Format: PNG
Deflectometry
This measurement method allows the assessment of reflective surface properties. PAINT contains strip-pattern deflectometries of the power plant's heliostats, enabling identification of surface deformations.
Format: HDF5
Kinematic
Mechanical arrangement and angles of the heliostat motors.
Format: JSON
Weather
Medium-quality data from plant-local sensors and high-quality data from the DWD weather station in Aachen-Orsbach. Measured quantities include short- and long-wave radiation, global surface irradiance, sunshine duration, cloud cover, humidity, vapor pressure, and visibility range.
Format: HDF5
